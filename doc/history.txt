TABLE task_history (
    task_id int
    user_id int
    event_id smallint
    notabene text
)

События:
    Иван Говно создал задачу

    Изменение параметров задачи:
        Иван Говно назначил исполнителя [Василий Блевотный]
        Иван Говно сменил исполнителя с [Василий Блевотный] на [Анжела Шлюхина]

        Иван Говно изменил дедлайн с [%s] на [%s]
        Иван Говно изменил важность с [%d] на [%d]

    Изменение статуса:
        Василий Блевотный взял задачу в работу (todo -> progress)
        Василий Блевотный поставил задачу на паузу (* -> pause)
        Василий Блевотный продолжил работу над задачей (pause -> progress)
        Василий Блевотный отправил задачу на проверку (* -> review)
        Василий Блевотный закончил задачу (todo -> done, progress -> done)
        Иван Говно принял решение задачи
        Иван Говно отправил задачу на доработку
        Иван Говно отменил задачу

        -> Иван Говно установил статус %s
